# Credit Card Fraud Detection

## Objective

The goal of this project is to detect fraudulent credit card transactions using Machine Learning.

## Dataset

The dataset contains:

- Time
- Amount
- V1 to V28 Features
- Class (0 = Genuine, 1 = Fraud)

Total Records: 284,807

Fraud Cases: 492

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm

Random Forest Classifier

## Steps Performed

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Scaling
5. Train-Test Split
6. Model Training
7. Prediction
8. Evaluation
9. Visualization

## Result

Accuracy Achieved: 99.95%

The model successfully identifies fraudulent credit card transactions.
